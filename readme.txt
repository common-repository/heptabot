Installez un ChatBot sur votre site avec WP-Heptabot. Créez un compte, gérez vos questions réponses, profitez de réponses complémentaires aux vôtres.


== Description ==

WP-HeptBot est un plugin Wordpress qui permet de mettre en place un ChatBot en quelques minutes sur votre site. Ce ChatBot entièrement gratuit vous permettra de répondre 24h / 24 à tous vos utilisateurs de façon automatique. en créant un compte heptabot, vous allez pouvoir : 
- gérer et éditer votre propre base de données de questions réponses
- suivre en temps réel les questions et les interactions avec vos utilisateurs
- activer ou non des questions réponses complémentaires sur une thématique précise ou générique.


== Installation ==

1. Il suffit de copier coller le répertoire wp-heptabot dans votre répertoire plugin sur votre serveur.
2. Activer ce plugin dans votre admin Wordpress
3. Aller sur la page de présentation du plugin et créer un compte heptabot en se rendant sur l\'url donnée
4. Installer le shortcode fourni sur votre site WP
5. Commencer à créer vos propres questions réponses.

== Frequently Asked Questions ==

= Ce plugin est-il gratuit ? =

Oui il est gratuit. Une version premium sera prochainement disponible avec des fonctionnalités supplémentaires

= Quelle différence entre mes questions / réponses et les questions génériques ? =

Votre ChatBot va prendre en priorité les questions / Réponses que vous lui aurez ajoutées. Mais comme vos utilisateurs vont sûrement chercher à vous coller et que vous ne pouvez pas répondre à tout, nous avons prévu une surcouche de questions réponses qui permettent de répondre à un grand nombre de questions plus génériques. Cet accès est activable ou pas. Libre à vous de l’utiliser ou pas.
